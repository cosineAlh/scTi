# 🛰️ scTi - 信息学院研究生人格内核扫描协议

> **MBTI 已过时，scTi (Information Science Personality Test) 才是属于信院搬砖人的底层协议。**

  

## 📋 项目简介

`scTi` 是一个专门为信息学院研究生设计的“图一乐”人格测试。

本项目基于原版 SBTI 的逻辑架构。无论你是发了顶会的 SOTA 战神，还是在实验室蹭空调的幽灵，这里都有属于你的逻辑位。

## 🛠️ 核心架构 (Kernel Architecture)

本项目通过 15 个学术维度对用户的电子魂魄进行全量扫描：

  - **自我模型 (Self)**: 学术自尊 (S1)、选题清晰度 (S2)、学术追求 (S3)
  - **情感模型 (Emotion)**: 导师依恋 (E1)、实验室投入度 (E2)、科研边界感 (E3)
  - **态度模型 (Attitude)**: 学术世界观 (A1)、实验规则感 (A2)、论文意义感 (A3)
  - **行动驱力 (Drive)**: 产出动机 (Ac1)、Debug 风格 (Ac2)、死线模式 (Ac3)
  - **社交模型 (Social)**: 交流主动性 (So1)、同门边界感 (So2)、组会真实度 (So3)

## ✨ 功能亮点

  - **全自动化炼丹 (Logic)**: 采用向量距离算法，精准匹配 27 种预设学术人格。
  - **数据可视化 (Visualization)**: 基于 **Chart.js** 渲染五维雷达图，直观展示你的学术内核负载。
  - **隐藏分支 (Easter Eggs)**:
      - 触发“怀疑世界真实性”选项将激活高维文明模拟器质问。
      - 内置 `sudo rm -rf` 逻辑自毁倾向检测。
  - **随机备注 (System Log)**: 动态生成诸如“显存监测：焦虑水平过载”等系统备注。
